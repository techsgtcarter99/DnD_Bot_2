***********************************WELCOME TO MY DUNGEONS & DRAGONS DICORD BOT***********************************

_________________________________________________________________________________________________________________

-This project's current status: NOT BEING WORKED ON AT THIS TIME
-Uses the Discord library to communicate with Discord servers
-Uses random integers from the random library for dice rolling and treasure picking
-Displays pages from Monster Manual for various monsters (has it's Table of Contents as well)

_________________________________________________________________________________________________________________

To call on the dice roll: #d# (1st # is for the number of dice you want to roll and second # is for the sided dice)
To call a monster sheet up: ?name (the name is for the name of the monster)
